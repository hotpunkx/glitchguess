# GLITCHGUESS 1v1 Multiplayer - User Guide

## Quick Start

### How to Play with a Friend

**Step 1: Create a Game**
1. Go to GLITCHGUESS homepage
2. Click **"1v1 MULTIPLAYER"** button
3. Enter your name
4. Click **"CREATE GAME"**

**Step 2: Share the Link**
1. You'll see a 6-character game code (e.g., "ABC123")
2. Copy the short link: `yourdomain.com/play/ABC123`
3. Send it to your friend via:
   - Text message
   - WhatsApp
   - Discord
   - Email
   - Any messaging app

**Step 3: Wait for Friend**
- Your friend opens the link
- They enter their name and join
- You'll see a notification when they join

**Step 4: Play!**
- One player thinks of a word (randomly assigned)
- The other player asks yes/no questions
- Try to guess within 20 questions
- Have fun! 🎉

## Game Roles

### Thinker (Secret Word Holder)
**Your Job:**
- Think of any object, person, animal, movie, or place
- Enter it as your secret word
- Answer questions with Yes, No, or Sometimes
- Be honest with your answers!

**Tips:**
- Choose something not too easy, not too hard
- Be consistent with your answers
- Don't give away hints in your answers

### Guesser (Question Asker)
**Your Job:**
- Ask smart yes/no questions
- Narrow down possibilities
- Make a final guess before 20 questions

**Tips:**
- Start with broad categories (Is it alive? Is it bigger than a car?)
- Use process of elimination
- Save your final guess for when you're confident

## Game Flow

### 1. Waiting Room (Player 1)
- You create the game
- See your game code
- Wait for opponent to join
- Can cancel if needed

### 2. Join Screen (Player 2)
- Open the shared link
- See Player 1's name
- Enter your name
- Click "JOIN GAME"

### 3. Game Starting
- Player 2 waits while Player 1 sets secret word
- Both players see "GAME STARTING..." message
- Game begins automatically when ready

### 4. Active Gameplay
**Thinker sees:**
- Their secret word (always visible)
- Questions from guesser
- Yes/No/Sometimes buttons
- Question history

**Guesser sees:**
- Question input field
- Submit button
- Answers from thinker
- Question history

### 5. Game End
- Game ends when:
  - Guesser guesses correctly (WIN!)
  - 20 questions reached (LOSE)
- Both players see results
- Secret word is revealed
- Option to play again (rematch)

## Rematch System

### How Rematch Works
1. Game ends with results screen
2. Either player can click "REQUEST REMATCH"
3. Other player sees the request
4. When both agree, new game starts
5. **Roles automatically switch!**
   - If you were thinker, you become guesser
   - If you were guesser, you become thinker

### Rematch Benefits
- No need to create new game
- Instant start
- Fair play with role switching
- Keep playing with same friend

## Tips & Tricks

### For Thinkers
- ✅ Choose concrete, well-known things
- ✅ Be consistent with your answers
- ✅ Think about how you'd answer before the question comes
- ❌ Don't choose obscure things nobody knows
- ❌ Don't change your mind mid-game

### For Guessers
- ✅ Start with category questions (animal? object? person?)
- ✅ Use binary elimination (bigger/smaller, living/non-living)
- ✅ Take notes if needed
- ✅ Save final guess for when confident
- ❌ Don't waste questions on wild guesses
- ❌ Don't ask the same question twice

### Communication
- Be respectful and have fun
- No cheating (don't look up answers)
- Be honest with your answers
- Enjoy the challenge!

## Troubleshooting

### "Game not found" Error
- Check the link is correct
- Make sure game code is typed correctly
- Game might have expired or been deleted

### "Game already started" Error
- This should no longer happen! (Fixed)
- If you see this, refresh the page
- Contact support if issue persists

### Friend Can't Join
- Make sure you sent the correct link
- Check they're using the full URL
- Try copying the link again
- Verify game is still in waiting status

### Page Refresh
- Don't worry! Your game is saved
- Refresh the page if needed
- You'll rejoin automatically
- Game continues where you left off

### Connection Issues
- Check your internet connection
- Refresh the page
- Both players need stable internet
- Real-time updates require connection

## Best Practices

### Good Secret Words
- ✅ Lion (animal)
- ✅ Pizza (food)
- ✅ Eiffel Tower (landmark)
- ✅ Harry Potter (character)
- ✅ Guitar (object)

### Avoid These
- ❌ Your neighbor's cat's name
- ❌ Obscure historical figures
- ❌ Made-up things
- ❌ Inside jokes only you know
- ❌ Abstract concepts

### Good Questions
- ✅ "Is it alive?"
- ✅ "Is it bigger than a car?"
- ✅ "Can you eat it?"
- ✅ "Is it a person?"
- ✅ "Does it have wheels?"

### Avoid These Questions
- ❌ "Is it exactly 5 feet tall?"
- ❌ "Does it start with the letter M?"
- ❌ "Is it my favorite thing?"
- ❌ Questions with multiple parts
- ❌ Questions that aren't yes/no
